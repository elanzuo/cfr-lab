# Kuhn Poker 规则（OpenSpiel 标准）

本文档描述 Kuhn Poker 的规则，数值与 OpenSpiel 实现保持一致（`kuhn_poker.h/.cc`）。

## 1. 核心参数

- **玩家数**：2～10（默认 2）。
- **牌堆**：`玩家数 + 1` 张牌，牌面编号为 `0..N`，数值越大越强。
- **底注**：每局开始前每人各下 1，底池初始为 N。
- **下注规模**：固定为 1，且**只有一轮下注**。

## 2. 动作编码（OpenSpiel 标准）

| 编码 | 枚举名 | 语义        |
| ---- | ------ | ----------- |
| 0    | Pass   | 过牌 / 弃牌 |
| 1    | Bet    | 下注 / 跟注 |

说明：

- 全程只有 **Pass/Bet** 两种动作，没有单独的 Fold。
- 若已有玩家下注，此时 **Pass 表示弃牌**，Bet 表示跟注。

## 3. 完整流程

1. **发底注**：所有玩家各下 1。
2. **发牌（Chance）**：依次给 Player 0、1、…、N-1 各发 1 张牌。
3. **下注轮**：从 Player 0 开始，依次行动（Pass 或 Bet）。
4. **若无人下注**：每人只行动一次，直接摊牌。
5. **若有人下注**：该玩家记为 `first_bettor`，其余玩家依次行动，直到轮到 `first_bettor` 之前为止（在 `first_bettor` 之前的玩家需要再行动一次）。无人可以再加注，只有 Pass 或 Bet。
6. **摊牌与胜负**：
   - 若无人下注：拥有**最高牌**的玩家获胜。
   - 若有人下注：只在**下注/跟注**的玩家中比较牌面，最高牌者获胜。

## 4. 收益定义

- 下注玩家总投入为 2（底注 1 + 下注 1）；未下注玩家总投入为 1。
- 终局收益定义为：**赢家获得底池减去自己投入**，其他玩家为其相反数。
  - 等价于 OpenSpiel 实现：赢家收益 `pot - bet`，其他玩家收益 `-bet`。

## 5. 常见误区提示

- Kuhn Poker **只有一轮下注**，下注额固定为 1。
- OpenSpiel 动作编码为 **Pass=0，Bet=1**。
- 多人版本中，若有人下注，**只有下注/跟注的玩家参与比牌**。
